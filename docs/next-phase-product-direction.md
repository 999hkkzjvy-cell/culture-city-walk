# 城市伴游优先方向

最后更新：2026-07-23

## 产品收拢

Cultural Citywalk 的下一阶段不再以“尽可能多的规划功能”为中心，而是优先做好一段可在现场独立完成的城市文化伴游：路线有明确主题，讲解有资料来源，用户知道下一步该做什么。

首轮只聚焦南京，准备三条人工审核的编辑稿：近代南京、民国建筑与城市生活、文学与书店；另有“人间烟火：一座城市的日常”作为固定日期的现场验证版。前三条在探索页标记为“编辑稿待审核”，不能被视为已发布的正式路线；第四条已写入本地正式路线包，可直接进入阅读和途中模式，但在部署及真实走测通过前不视为永久发布。

2026-07-23 起，探索页的编辑稿可打开独立审核页，集中展示主题主线、4–6 个核心文化站及其角色、可追溯资料链接与发布前待核验项。民国建筑稿已用杨廷宝住宅（外观观察）和童寯故居与童寯建筑馆替换拉贝故居；书店稿保留朴阅书店与先锋五台山总店，并加入学人书店与方所。“人间烟火：一座城市的日常”保留 0725 的固定时间表，并把细分观察点整理为 13 个独立站：水西门广场与三山门遗址碑、童寯故居与童寯建筑馆各合为一站；早餐、午餐作为服务站，也分别保留深读线索和任务方向；交通仍只作为时间安排，不参与主题排序。它使用独立正式路线包和预置离线导览卡，可直接开始现场伴游；旧 demo 中属性含糊的“五台山体育馆旧址”“顺和路公馆区”未写入精选路线内容包。

## 已实现的基础

- 首页和探索页聚焦三条南京主题编辑稿及一条现场验证版；主导航收拢为探索路线、规划路线、我的路线。
- 三条编辑稿及现场验证版均建立结构化资料台账：官方/权威资料支持历史与场馆事实，地图资料只支持地点信息；每个站点都声明自己在路线中的叙事角色和待审核项。
- 规划页默认只生成 5–6 个可调整项；餐厅最多保留 3 个候选，并在缺少高德评分时明确说明。
- 途中模式在移动端改为单列阅读，路线、深读、任务、照片和下一步行动可离线读取；已生成深读会写入本地缓存。
- 站点深读使用百度 AI 搜索三组检索，再由 DeepSeek 依据资料生成；页面会显示检索次数、返回量和采用量。
- 现场问答要求登录，沿用同一份站点资料和来源展示；问答记录只存在当前设备。

## 深读质量规则

- 提示词版本为 `route-collaboration-v0.8`，角色为城市文化导游，而非虚构亲历的朋友；有明确发布主体的资料可用于一般城市史、街区沿革、人物生平和日常生活叙述，并附链接供读者追溯。
- 资料充分时，普通文化站为 800–1200 字：摘要加四段导览，分别覆盖历史背景、现场空间、人物或人文故事、与今天城市的联系；没有独立历史沿革的餐饮休息站不少于 500 字。
- 官方、学术、权威媒体及经过平台审核的百科词条，以及具明确发布主体的城市文化资料，均可支持一般历史、生平和建筑事实；动态开放、价格、安全以及战争伤亡、暴力、慰安所、强占等严重指控仍须官方、档案、学术或路线编辑逐项确认。
- 检索资料不足时生成较短的资料有限版，不以编造内容补足长度。
- 每站固定两项可选现场任务，任务不计分，也不以“闯关”命名。

## 人工审核清单

每条正式路线发布前，人工确认：

1. 4–6 个体验站点均为清楚、可抵达的文化站点；
2. 路线能用一句话解释主题主线；
3. 每个核心站点都有可访问的资料来源和审核日期；
4. 深读中的年代、人物、引语、传说和动态信息均符合来源等级；
5. 高德步行、开放时间、预约和休息点经过出发日复核。

## 小规模真实使用

邀请 5–10 位用户完成真实行走。通过条件：多数用户无需额外说明即可走完路线、愿意再次使用、能说出路线主题；出现严重事实、开放信息或安全问题时停止扩展并先修复。

暂不进入社交社区、路线交易、全城市自动覆盖、音频导览和第三方餐厅点评聚合。

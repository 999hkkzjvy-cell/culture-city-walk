# 项目提交日志

本文件记录 Cultural Citywalk 项目的提交内容和功能完善。以后每次提交前都要先更新本文件，并使用中文提交信息。

注意：`ignore-files/` 默认被 `.gitignore` 忽略；本文件需要单独强制纳入版本控制，其他参考文件仍保持忽略。

## 提交规范

- 提交信息使用中文，简洁说明本次完成的功能或修复。
- 每次提交前更新本日志，新增一条包含提交目的、主要改动、验证情况的记录。
- 同步维护 `docs/ai-project-context.md`，让后续 AI 优先阅读后即可理解当前项目状态。
- 不把 `.env.local`、Supabase service role、DeepSeek API Key、高德 Web Service Key 等密钥写入提交。

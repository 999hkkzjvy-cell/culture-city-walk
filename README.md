# Citywalk AI

一个以城市文化、主题滤镜和路线时间轴为核心的私人城市漫游助手。

## 当前阶段

阶段 1：项目基础与视觉骨架。

- Next.js App Router + TypeScript
- 静态导出，适配 GitHub Pages
- 首页、规划页、路线阅读页
- localStorage 草稿保存与恢复
- Vitest 纯函数测试
- Playwright 桌面与移动端冒烟测试
- GitHub Pages workflow

## 本地开发

```bash
npm install
npm run dev
```

## 验证

```bash
npm test
npm run build
npm run e2e
```
